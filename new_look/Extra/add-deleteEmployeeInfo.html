Follow these steps

Step 1.

- Go to the Request Form Responses spreadsheet -> Here

If adding an employee follow steps 2 - 15, if deleting an employee follow steps 2a - 15a

Step 2.

- Click on the red Add New Employee Button

Step 3.

- Type in the persons First Name and click OK

Step 4.

- Type in the persons Last Name and click OK

Step 5.

- Wait about 30secs to a minute for the system to add the new tables. A pop up should show saying "HTML file with updated iframes created in your Drive folder." click ok, Then a pop up should show saying "Employee tables created and dropdown links updated! " click ok.

Step 6.

- Navigate to the HTML GEN files folder -> Here

Step 7.

- Find the file of the persons name you just created. For example if the persons name is John Doe find the file name John_Doe.html and download the file

Step 8.

- Find the file Requests_Dropdown_Button.html and download the file

Step 9.

- Navigate to Github (You'll need to create a github account if you don't already have one. Then message Axel:(903)503-4942 Your Github Name to give you permission) -> Here

Step 10.

- Click on the new_look folder, then click on the crew folder

Step 11.

- Click the Add file button on the top, Then Click Upload files

Step 12.

- Drag and drop the person name file here, For example John_Doe.html, then click Commit changes. Changes can take up to 5 minutes

Step 13.

- Navigate back to the root website folder, and click menu_buttons

Step 14.

- Click the Add file button on the top, Then Click Upload files

Step 15.

- Drag and drop the Requests_Dropdown_Button.html file here, then click Commit changes. Changes can take up to 5 minutes

Follow these steps to delete an employee

Step 2a.

- Click on the white Remove an Employee button

Step 3a.

- Type in the persons First Name and click OK

Step 4a.

- Type in the persons Last Name and click OK

Step 5a.

- Wait about 30secs to a minute for the system to delete the tables. A pop up should show saying "Employee tables and HTML file deleted, and dropdown links updated!" click ok.

Step 6a.

- Navigate to the HTML GEN files folder -> Here

Step 7a

- Find the file Requests_Dropdown_Button.html and download the file

Step 8a.

- Navigate to Github (You'll need to create a github account if you don't already have one. Then message Axel:(903)503-4942 Your Github Name to give you permission) -> Here

Step 9a.

- click on the menu_buttons folder

Step 10a.

- Click the Add file button on the top, Then Click Upload files

Step 11a.

- Drag and drop the Requests_Dropdown_Button.html file here, then click Commit changes. Changes can take up to 5 minutes

Step 12a.

- Navigate back to the root website folder, then click on the new_look folder, then click on the crew folder
Step 13a.

- Find the employee name you just deleted for example John_Doe.html, then click on the file

Step 14a.

- Click on the three dots on the top ...

Step 15a.

- Click on delete file, then click on commit changes at the top. Changes can take up to 5 minutes
