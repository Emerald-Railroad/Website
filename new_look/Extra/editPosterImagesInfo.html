Website Help

Step 1.

- Navigate to the Website Sheets folders -> Here

Step 2.

- Download the file poster_images.docx and then open it in Office 365 Word

Step 3.

- Delete any rows that contain old movies. (Highlight the entire row you want to delete, click on the delete dropdown, then click delete rows)

Step 4.

- Add New Rows, (Right Click on the row below where you want to add a new row, click the insert dropdown, then click insert row above)

Step 5.

- In the white column, Type the movie name, then underneath the name type the Release date of the movie.

Step 6.

- Add photos of the images for posters A,B or C in the corresponding columns. (to add a photo, copy an image from another poster row then paste in the box where you want the new image for the new poster. Then, right click the image, click change picture, click This device and find the picture you want to replace the old one with. Click ok)

Step 7.

- Repeat step 6 for all new posters

Step 8.

- After All old poster have been deleted and new ones added, Make sure they are in alphabetical order from the top down.

Step 9.

- to use these for the website and connect them to one big photo, we need to screen shot each table on this document. To screenshot hold down the Windows key, Shift Key and the S button. A plus icon appears, drag your mouse from the top corner of the table (excluding the white bars on the top and bottom of the page) down to the bottom right corner. Afterwards a notification will appear on your PC, click on it. then click on the save icon on the top. Change the file name to "poster_images_page-0001.jpg" and save to your folder. Repeat this step for all pages in the document but for each new image add a number to the file name, for example, poster_images_page-0001.jpg, poster_images_page-0002.jpg..., poster_images_page-0012.jpg

Step 10.

- Navigate to Github (You'll need to create a github account if you don't already have one. Then message Axel:(903)503-4942 Your Github Name to give you permission) -> Here

Step 11.

- Click on the images folder

Step 12.

- Click the Add file button on the top, Then Click Upload files

Step 13.

- Drag and drop the picture files here, these will replace the exsisting images so make sure you named your files correctly. then click Commit changes. Changes can take up to 5 minutes
