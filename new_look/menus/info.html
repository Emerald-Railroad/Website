Poster Rules

  • You can request a poster once per week and up to four posters per month

  • One request can be made, each time the form is submitted

  • Requesting a poster does not guarantee you will receive it

  • Posters appear two months at a time (currently accepting requests for December, January and currently showing movies)

  • All requests are stored on the website under the view requests tab

  • Requests can be viewed by clicking on "View Requests" tab and then clicking your name

  • You only have to request a poster once (avoid requesting the same poster more than once)

  • You can only request the same type of poster once, duplicates will not be handed out unless extras are available after everyone who wanted one has received one.

  • The leadership team (Supervisors and Managers) have first priority on requests, after that posters are handed out in order of who requested.

Weekly Request Limit Resets

  • You may only request up to one poster per week

  • Request limits reset every friday and poster options reset at the first of each month

  • The name dropdown list includes everyones name

  • Choose your name when making a request

  • When you make a request your name will automatically be removed from the dropdown. It will reappear on friday when the weekly reset activates

Requests: Current Month vs Awaiting?

Current Month

- These are the requests that were made in the current month

- These Requests will appear in the red table when viewing your requests

Awaiting Request

- These are the requests that were made in a previous month and you are still waiting to recieve them

- These Requests will appear in the gray table when viewing your requests
