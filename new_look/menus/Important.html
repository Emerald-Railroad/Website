Important New Changes

    Welcome to Version 8.0 of the website, With this update comes some major changes

    The Poster Request System has been rebuilt from the ground up. This fixes the small bugs with the old version (7.4)

    Requests are still limited to once per week, however a new system has been put in place to enforce this.

    - Previously after making a request your name would be removed from the form for the rest of the week. With the New system your name will no longer be removed. This means one could put in multiple requests per week however, everything is done in the backend now. For example, you can put in one request and the system will record your response, if you submit another response in the same week, the system will automatically delete any new responses until a new week has begun.

    - The system still resets on each Friday

    - A New Bar has been added to the top of the request form page, This will show the date of the next weekly reset and when you can submit any new requests.

    - The New Bar should help when finding out when you can request again

    - With this new system, it resets automatically. The old system needed to be reset manually each week by Axel. That is no longer needed here and is the main reason for this change

    - Overall the system will behave the exact same its just being excuted in a different way
