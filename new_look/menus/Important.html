New Inventory Page

    New Exciting Changes made to the inventory table

    New Collums have been added!

    - The Currently Requested Collum! The Currently Requested Collum will now show you how many associates have requested that current poster. Eliminating the need to view each person's individual poster table.

    - The Posters Remaining collum! The Posters Remaining collum will show you how many posters are currently left to be requested.

    New Colors!

    - Each collum now has its own separate color. The Total Poster collum is red, Currently Requested collum is Orange and the Poster Remaining collum is Yellow. The Poster A, B, and C collums now color match the poster images tab.
